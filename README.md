##  Documentation
This module builds basic JavaScript array manipulation methods from scratch.

These array methods include ```push()```, ```pop()```, ```shift()```, ```unshift()```, ```forEach()```, ```map()```, ```filter()```, ```reduce()```, and ```slice()```.
